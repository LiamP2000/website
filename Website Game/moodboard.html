Design Homepage

Ik heb voor het lettertype 'Chalkduster' gekozen omdat ik het een speels lettertype vond en de kleine kapjes in de letters mij doen denken aan boomschors.

Javascript plugin per webpage

Op mijn 'home' page heb ik de titel laten opkomen, hiervoor heb ik gebruik gemaakt van javascript. Het geeft een soort fade-in effect.

Op mijn 'game' page heb ik de de scrollreveal library gebruikt. Dit zorgt ervoor dat het een soort effect geeft als je door de pagina scrollt.

Ook heb ik hier een validation voor mijn form gemaakt zodat er een melding gegeven wordt als het geen geldig e-mailadres is of het '@' teken ontbreekt.

Op mijn 'How does it work?' page heb ik de titel laten verschijnen door middel van javascript. Ik heb alle letters apart laten verschijnen zodat dit een leuk effect geeft als je de pagina bezoekt.
